## 概要

スーパーマーケットに置いてある様な食材を購入します。

食材を買い物カゴに入れたり、削除をすると合計金額が自動計算されます。

ページネーション、並べ替え（金額）、食材の種類により検索機能も付いています。

バックオフィス機能として食材の追加や金額の修正、削除などのCRUD操作も可能です。
