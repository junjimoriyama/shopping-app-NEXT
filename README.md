## 概要

スーパーマーケットで販売されているような食材を購入できます。
買い物カゴに食材を追加・削除すると、合計金額が自動で計算されます。

また、以下の機能を備えています。

ページネーション（一覧を複数ページに分割）
並べ替え機能（価格順など）
検索機能（食材の種類でフィルタリング）
さらに、バックオフィス機能として、食材の追加・価格の修正・削除などのCRUD操作が可能です。
